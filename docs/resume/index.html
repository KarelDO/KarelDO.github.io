NLP Internship

Jul 2021 - present

Waylay, Belgium

I designed and built a machine learning-based semantic parser which allows the user to interface with the Waylay system over natural language.

This problem was particularly challenging because of the extremely low amount of data available. I solved this by leveraging large pre-trained language models (specifically, GPT3) together with prompt engineering to build an extremely data-efficient semantic parser to translate natural language inputs into a format digestable by the Waylay system.
